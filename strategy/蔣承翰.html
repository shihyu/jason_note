交易不是比誰賺得多，交易是「比誰活得久」

短線交易

  1. 價差才是真理 1~2%, 3~4%
  2. 高週轉率 | MGK:「富人靠報酬率，窮人靠週轉率」
  3. 商品的選擇 | e.g. 買現股會扣息

權證 => 不會被扣二代健保，10 倍槓桿，除權息後吃的到「溢價」

  1. 找會動的標的 不要玩死魚 (e.g. 中華電 => 1~2 ticks)

交易的流派

  1. 當沖 => 開盤流, tick流,
  2. 隔日沖 => 族群聯想力, 鎖漲停
  3. 短波段 => 事件, 技術面, 產業消息, 黑暗之力

短線交易需要具備的東西 (物理上)

  1. 設備 主要是網速 (雙螢幕就夠了 => 太多會變成「雜訊太多」)
  2. 最理想是有交易室 => 跟日常生活分離
  3. 好營業員 (資訊量, 處理速度, 手續費, 融資利率 => 交易量大的時候影響很大, 帥哥正妹)
  4. 給你支持，救你心態的人 => 交易的夥伴 (證券研究社 前社長 許俊翔)

短線交易需要具備的東西 (心理上)

  1. 穩定的心態，不以物喜、不以己悲（面對自己的損益要檢視錯誤，看到別人賺錢不要過度羨慕，思考邏輯）
  2. 抑制自己的貪婪與恐懼
  3. 忌諱不認輸 ➡️ 「永不攤平」: 否則損益結構會變成: 小賺+小賠+大賠 ➡️ 0.9 * 1.1 < 1
  4. 留得青山在，不怕沒柴燒

操作績效

  1. 17 個月不賠錢
  2. 2 個月震盪, 賠了 37 萬
  3. 之後績效又開始忘上跑

學習短線交易

  1. 短線交易一言難盡，沒練習就開始 = 散財童子
  2. 基礎知識，教科書書籍，主管機關網站
  3. 模擬交易 XQ 全球贏家 ➡️ 「本業」為重！ ➡️ 建議：「先在模擬單穩定獲利，再來談實單交易」
  4. 一開始別拿太多錢，試試看
  5. 成功很好，慢慢放大
  6. 失敗沒關係，交易不是唯一

家庭要求

  1. 家庭成員都「沒有負債」
  2. 自己的本金先累積到 15 萬

交易邏輯

  1. 找市場喜歡 or 討厭的東西
  2. 市場的舔狗
  3. 大家都忘記，但未來會想起它們的好➡️ 低基期、轉機股：業績、法說會
  4. 盤感（經驗與直覺）➡️ 交易複盤 ("大幅震盪股" 前 100 名)
  5. 市場「情緒反轉」的力量
  6. 部位控管 ➡️ 最重要

題材面利多利空 開盤當沖

  1. 題材 => 下午 4~6 點公告，晚上 10 點看工商時報、鉅亨網、MoneyDJ、媒體新聞
  2. 判斷 => 09:00~09:10，試搓價量，五檔變化，往內盤丟股票
  3. 下單

緩搓盤

  1. 下單 => 方向明確實，於「緩搓盤期間」掛單
  2. 挑選 => 成交量大且內外盤皆積極、適當的胃納量

事件

  1. 記者會、近期會有利多
  2. 新聞發布前「蹲點」

    口罩股、廷鑫等
