高報酬 ETF 實戰組合手冊

（SMH / USD / SOXL / QLD / SPMO）

目標：在可控回撤下，最大化趨勢段報酬
適合：熟悉趨勢、動能、願意使用 2× / 3× ETF 的投資人

一、ETF 倍率速查表

ETF倍率標的定位

USD2×費城半導體（SOX）主力
SOXL3×費城半導體（SOX）加速
QLD2×Nasdaq-100科技主線
SPMO1×S&P500 動能因子市況濾網

二、最佳實戰組合（推薦）

組合 A｜核心＋倍率滑桿（最穩定）

SMH + USD + SOXL

使用邏輯

  • 震盪 / 打底：SMH
  • 趨勢初期：USD
  • 主升段噴出：SOXL
  • 回檔不破趨勢：SOXL → USD

重點：2× 是主力，3× 只在趨勢段使用

組合 B｜雙引擎（半導體＋科技）

SMH + USD + QLD

使用邏輯

  • 半導體為景氣與 AI 主線
  • Nasdaq 為資金主線
  • 避免單一產業熄火風險

適合：不想天天看盤，但仍希望放大報酬

組合 C｜動能濾網進階版

SPMO + USD + SOXL

使用邏輯

  • SPMO 強於 S&P500 → 市場為趨勢盤
  • 只在趨勢盤開 2× / 3×
  • SPMO 轉弱 → 槓桿降回 1×

不預測，只判斷「值不值得開槓桿」

三、倍率使用原則（很重要）

為什麼要 2×？

  • 2× 是「可以持有」的上限
  • 震盪盤仍能存活
  • 報酬已顯著放大

3× 的正確用法

  • 只用在：
    • 明確趨勢
    • 突破後主升段
  • 不用在：
    • 打底
    • 高檔震盪
    • 系統未確認前

四、實戰一句話總結

  • ❌ All in SOXL = 遲早畢業
  • ✅ SMH（底）＋ USD（力）＋ SOXL（爆）= 活得久又賺得多
  • ✅ 槓桿不是常駐，是「條件成立才打開」
