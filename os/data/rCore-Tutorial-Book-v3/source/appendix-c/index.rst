附錄 C：深入機器模式：RustSBI
=================================================

.. toctree::
   :hidden:
   :maxdepth: 4

RISC-V指令集的SBI標準規定了類Unix操作系統之下的運行環境規範。這個規範擁有多種實現，RustSBI是它的一種實現。

RISC-V架構中，存在著定義於操作系統之下的運行環境。這個運行環境不僅將引導啟動RISC-V下的操作系統， 還將常駐後臺，為操作系統提供一系列二進制接口，以便其獲取和操作硬件信息。 RISC-V給出了此類環境和二進制接口的規範，稱為“操作系統二進制接口”，即“SBI”。

SBI的實現是在M模式下運行的特定於平臺的固件，它將管理S、U等特權上的程序或通用的操作系統。

RustSBI項目發起於鵬城實驗室的“rCore代碼之夏-2020”活動，它是完全由Rust語言開發的SBI實現。 現在它能夠在支持的RISC-V設備上運行rCore教程和其它操作系統內核。

RustSBI項目的目標是，製作一個從固件啟動的最小Rust語言SBI實現，為可能的複雜實現提供參考和支持。 RustSBI也可以作為一個庫使用，幫助更多的SBI開發者適配自己的平臺，以支持更多處理器核和片上系統。
