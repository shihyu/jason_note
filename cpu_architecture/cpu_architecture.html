CPU 架構技術

本章節介紹現代 CPU 架構技術，特別聚焦於當前主流與新興的處理器架構設計。

CISC vs RISC

  • CISC (Complex Instruction Set Computer)：複雜指令集，以 x86/x86_64 為代表
  • RISC (Reduced Instruction Set Computer)：精簡指令集，以 ARM 和 RISC-V 為代表

為什麼關注 ARM 和 RISC-V？

產業趨勢

  1. 能源效率：行動裝置與資料中心都需要更高的能效比
  2. 開放生態：RISC-V 的開源特性帶來新的商業模式
  3. 客製化需求：特定應用場景需要針對性的架構設計
  4. 地緣政治：技術自主與供應鏈安全考量

技術優勢

  • ARM：成熟的生態系統、優秀的功耗控制、廣泛的工具鏈支援
  • RISC-V：開放標準、模組化設計、無授權費用、易於客製化
