\frametitle{单用户系统(1945-1955)}


\begin{itemize}
\item 手动连线/纸带传输进行程序输入
\item 机器成本远大于人力成本	
\item 操作系统=装载器(loader)+程序库(libraries) 
\item 问题：昂贵组件的低利用率
\end{itemize}
