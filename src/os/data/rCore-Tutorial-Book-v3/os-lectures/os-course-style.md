# 遠程上課方式
## 課前
* 鼓勵課前預習：瀏覽下次課[視頻和](http://os.cs.tsinghua.edu.cn/oscourse/OS2020spring/#A.2Bi.2F56C4uyTkk-)[slides](http://os.cs.tsinghua.edu.cn/oscourse/OS2020spring/#A.2Bi.2F56C4uyTkk-)，以及相關的[課堂問題](https://github.com/LearningOS/os_course_exercises/blob/2020spring/SUMMARY.md)。
* 注意上課時間，提前幾分鐘登陸遠程會議系統（騰訊會議系統）和荷塘雨課堂，按時上課。
## 課中（0-95分鐘）
在課程安排上面，嚴格遵照校歷安排的時間，一週兩次，每次兩節45分鐘的課+5分鐘休息。具體細節如下:

### 60分鐘視頻學習+10分鐘相關簡單練習+5分鐘課間休息
* （0-65）前2分鐘，老師遠程會議（或微信）方式宣佈開始上課，老師通過雨課堂先發布課堂練習題目，學生選擇合適自己的方式(下載本地播放，學堂在線觀看等)進行視頻學習；同時回答課堂練習題目。在此期間，如有問題，可通微信大群或微信小群提問，複雜一些的問題儘量提在在線文檔上（石墨文檔等）。助教/老師收集問題，簡單的就直接微信文字回答或遠程會議方式回答，複雜一點的在課程後20分鐘的交流中交流討論。助教/老師也可根據情況在此期間通過微信或遠程會議方式與部分學生交流討論。
* （40-45）分鐘時，課間休息
* （65-75）分鐘左右時，老師點評課堂練習回答情況，並對同學遇到的共性問題進行解答。
### 20分鐘：小組討論
* （75-95）分小群討論，基於會議軟件/微信等，助教點評上次課的課後作業，回答學生比較複雜的問題，助教提問比較麻煩的問題，請學生回答。
## 課後
* 完成實驗和不定期的練習。
## 其它
* 如學生在60分鐘內看不完視頻，可繼續看視頻，不參加後續的老師點評和小組討論。
* 每週一第二大節的第三小節是大實驗組的討論。助教+老師參加。
