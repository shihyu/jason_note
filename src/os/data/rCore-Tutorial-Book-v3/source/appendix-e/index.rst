openEuler操作系統
------------------------------------

國產操作系統始於九十年代，大多數基於Fedora/CentOS/Debian/Ubuntu進行二次開發，直到2019年，華為公司開源了openEuler操作系統， openEuler是自主演進的根操作系統，不基於其他任何OS二次開發，這是與其他國產OS的主要差異。

openEuler是一個開源、免費的Linux發行版平臺，其致力於打造中國原生開源、可自主演進操作系統根社區。openEuler前身是華為公司發展近10年的服務器操作系統EulerOS，2019年開源，更名為openEuler，當前有多個國產OSV廠商基於openEuler發佈商用版本。openEuler最新定位是面向數字基礎設施的開源操作系統，支持多樣性計算，支持服務器、雲計算、邊緣計算、嵌入式等應用場景，支持OT（Operational Technology）領域應用及OT與ICT的融合。

openEuler在具有通用的Linux系統架構，包括內存管理子系統、進程管理子系統、進程調度子系統、進程間通訊（IPC）、文件系統、網絡子系統、設備管理子系統和虛擬化與容器子系統等。同時，openEuler又不同於其他通用操作系統，openEuler從OS內核、可靠性、安全性和生態使能等方面做了特性增強。openEuler的關鍵特性如下表所示：
