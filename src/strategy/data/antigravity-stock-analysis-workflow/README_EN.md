## ğŸš€ Quick Start

### 1. Full Analysis on a Single Stock
Use this when you want to conduct in-depth research on a stock:
```
/full-analysis [TICKER]
```
**Example**: `/full-analysis RCAT`
- Automatically collects real-time quotes, financial reports, institutional holdings
- Checks catalysts (PDUFA, earnings dates, contracts)
- Calculates risk score
- Provides entry/exit recommendations
- Auto-generates individual reports and updates summary

### 2. Short-term Scan
Use this when looking for potential trading opportunities:
```
/stock-scan
```
- Scans today's hot stocks, top gainers, unusual options activity
- Automatically filters out low-quality targets
- Generates a "Short-term Screening" report

### 3. Generate Summary Report
When you've analyzed multiple stocks and want to see aggregated results:
```
/summary
```
- Reads all Session data
- Compares priority and risk/reward ratios across all targets
- Creates a catalyst timeline
- Outputs final trading decision recommendations

## ğŸ’¡ Best Practices

1. **Daily Workflow**:
    - Run `/stock-scan` first for ideas
    - Select 3-5 interesting targets
    - Run `/full-analysis` on each target
    - Finally run `/summary` to create a daily report

2. **Key Focus Areas**:
    - **PDUFA/Earnings**: Use `/catalyst-check` to ensure accurate dates. Binary event risks are high for biotech.
    - **Risk Score**: 80+ is "High Priority", 60-80 is "Medium Priority", below 60 suggests watching.

3. **Data Verification**:
    - The system tries to fetch real-time data, but verify during market hours for current volatility.
