## [](#短線交易需要具備的東西-物理上 "短線交易需要具備的東西-物理上")短線交易需要具備的東西 (物理上)

1.  設備 主要是網速 (雙螢幕就夠了 => 太多會變成「雜訊太多」)
2.  最理想是有交易室 => 跟日常生活分離
3.  好營業員 (資訊量, 處理速度, 手續費, 融資利率 => 交易量大的時候影響很大, 帥哥正妹)
4.  給你支持，救你心態的人 => 交易的夥伴 (證券研究社 前社長 許俊翔)

## [](#短線交易需要具備的東西-心理上 "短線交易需要具備的東西-心理上")短線交易需要具備的東西 (心理上)

1.  穩定的心態，不以物喜、不以己悲（面對自己的損益要檢視錯誤，看到別人賺錢不要過度羨慕，思考邏輯）
2.  抑制自己的貪婪與恐懼
3.  忌諱不認輸 ➡️ 「永不攤平」: 否則損益結構會變成: 小賺+小賠+大賠 ➡️ 0.9 \* 1.1 < 1
4.  留得青山在，不怕沒柴燒

## [](#短線交易-重點 "短線交易-重點")短線交易 重點

1.  市場共識
2.  事件明確（的股票或期貨事件交易）
3.  逆向策略（找市場恐慌丟出的籌碼）  
    4.把輸很多的交易設成桌布深刻檢討  
    小心上頭  
    （上頭：主觀意識太強、太看好一檔股票  
    ，但市場不領情，法人丟籌碼給自己、散戶）  
    控制好人性衝動  
    區分上頭 vs 好機會
4.  尊重人性的瘋狂，與市場的趨勢  
    人性的瘋狂：  
    口罩股、廷鑫等  
    「基本面不佳」、「有下市風險」的股票  
    都還是有連續 9 根漲停的機會
5.  總資金上億後的前輩  
    6-1. 交易能力「更上層樓」：鎖漲停的股票  
    6-2. 「基本面、消息面」投資：提升生活品質、交易以外其它樂趣的時間
