## 核心交易哲學

### 反市場與情緒套利

**核心口訣**：
> 「平常不要買賣，利多大漲時出掉，利空大跌時買進」

**操作原則**：
- 利用市場的過度反應獲利
- 大盤大跌、融資斷頭時尋找買點
- 營收創新高、利多見報時獲利了結

### 生活化交易與耐心

**心態管理**：
- 將賺錢視為日常生活的一部分
- 保持從容不迫，在泡澡、旅遊時也能完成交易
- 保留高額現金部位，只在「送分題」時重壓

## 選股邏輯與標的特徵

### 1. 必備「股票期貨」

**特徵與優勢**：
- 90% 以上標的具備股票期貨
- 交易稅僅 1/150000，成本極低
- 高槓桿、資金效率高
- 多空切換靈活

### 2. 報價驅動與景氣循環股

**偏好族群**：
- **航運**: 運價指數 SCFI/BDI
- **記憶體**: DRAM/NAND 合約價
- **金屬/鋼鐵**: 鎳、銅國際報價

**選股理由**：
- 趨勢一旦形成，受國際市場推動
- 波動性大且消息具備延續性

### 3. 族群作戰策略

**打群架邏輯**：
不單押單一股票，而是操作整個板塊

**實例**：
- **PCB 戰隊**: 聯茂(CCL) + 尖點(鑽針) + 金居(銅箔) + 富喬(玻纖)
- **航運三雄**: 長榮、陽明、萬海
- **網通群**: 神準、智易

### 4. 高 Beta 值（股性活潑）

**選股特徵**：
- 專挑會漲停、會大跌的股票
- 避開牛皮股（電信、大型金融）
- 對台積電 (2330) 興趣缺缺

**詳細內容**: [XMZ交易選股與進出場全攻略](XMZ_Complete_Trading_Guide.md)

## 進出場戰術執行

### 進場策略

**買綠不買紅**：
- 堅持在股價拉回、修正時佈局
- 市場融資大減、跌停家數暴增時進場「撿屍」

**技術指標**：
- **60分K線**: 支撐位參考 20MA，最佳買點參考 60MA
- **缺口回補**: 關注跳空缺口的回補機會

### 出場策略

**利多出貨**：
- 消息見報大漲時出貨
- 「賺錢要簡單」原則

**分批獲利**：
```
第一批: 賣一半入袋為安
第二批: 留一半博取超額利潤
```

**時效性**：
- 進場後 3 天內未如預期發動或過高，則換股操作
- 不跟股票談戀愛

**放空機制**：
- 對過度炒作、基本面不符的股票進行放空
- 案例：中工、轉弱後的萬海

**詳細內容**: [XMZ交易選股與進出場全攻略](XMZ_Complete_Trading_Guide.md)

1. **心態超脫**
   - 不被市場情緒左右
   - 反向利用市場情緒

2. **效率工具**
   - 利用股期的高槓桿與低成本

3. **趨勢共振**

4. **果斷執行**

   - 買點不到敢等

## 🎯 操作檢查清單

### 進場前確認

- [ ] 標的具備股票期貨
- [ ] 屬於報價驅動或景氣循環產業
- [ ] 市場情緒處於恐慌階段（買入）
- [ ] 技術指標回測 60MA（最佳買點）
- [ ] 資金準備充足（高槓桿風險控管）

### 持倉中監控

- [ ] 國際報價趨勢是否延續
- [ ] 投信是否持續買超
- [ ] 消息面是否轉向
- [ ] 股價是否接近目標
- [ ] 是否出現利多見報（準備出貨）

### 出場後檢討

- [ ] 是否在利多時出貨
- [ ] 3天法則是否執行
- [ ] 停損是否果斷
- [ ] 族群輪動是否掌握
- [ ] 紀錄操作日誌
