# XMZ 交易選股與進出場全攻略 (完整彙整版)

本指南整合了 XMZ 的交易哲學、選股邏輯、技術進出場手段以及具體的操作標的分析，旨在呈現一套完整的「反市場情緒」交易系統。

---

## 一、 核心交易哲學 (Core Philosophy)

### 1. 反市場與情緒套利
*   **核心口訣**：「平常不要買賣，利多大漲時出掉，利空大跌時買進」。
*   **群眾心理**：利用市場的過度反應獲利。當大盤大跌、融資斷頭、市場一片哀嚎（如俄烏戰爭爆發、疫情嚴峻）時，是他尋找買點的時機。
*   **利多出盡**：當營收創新高、大利多見報，市場情緒達到最高點時，是他獲利了結的時刻。

### 2. 生活化交易與耐心
*   **心態**：將賺錢視為日常生活的一部分，而非壓力來源。在泡澡、旅遊、浮潛時完成交易，保持從容不迫。
*   **耐心**：保留高額現金部位，不隨意進出，只在看到「送分題」時重壓。

## 二、 選股邏輯與標的特徵 (Stock Selection Logic)

XMZ 的選股並非隨機，而是具有高度的一致性：

### 1. 必備「股票期貨 (股期)」
*   **特徵**：選中的股票 90% 以上具備股票期貨標的。
*   **原因**：成本極低（交易稅僅 1/150000）、資金效率高（高槓桿）、多空切換靈活。

### 2. 偏好「報價驅動」與「景氣循環股」
*   **族群**：
    *   **航運** (運價指數 SCFI/BDI)。
    *   **記憶體** (DRAM/NAND 合約價)。
    *   **金屬/鋼鐵** (鎳、銅國際報價)。
*   **理由**：這類產業趨勢一旦形成，受國際市場推動，波動性大且消息具備延續性。

### 3. 族群作戰 (打群架策略)
*   **邏輯**：不單押單一股票，而是操作整個板塊。
*   **實例**：
    *   **PCB 戰隊**：聯茂(CCL) + 尖點(鑽針) + 金居(銅箔) + 富喬(玻纖)。
    *   **航運三雄**：長榮、陽明、萬海。
    *   **網通群**：神準、智易。

### 4. 高 Beta 值 (股性活潑)
*   **特徵**：專挑會漲停、會大跌的股票。避開牛皮股（如電信、大型金融），對台積電 (2330) 興趣缺缺，認為那是外資控盤工具，非靈活操盤首選。

## 三、 進出場戰術執行 (Trading Tactics)

### 1. 進場策略 (Entry)
*   **買綠不買紅**：堅持在股價拉回、修正時佈局。
*   **恐慌買點**：市場融資大減、跌停家數暴增時進場「撿屍」。
*   **技術指標**：
    *   **60分K線**：支撐位參考 **20MA**，最佳買點參考 **60MA**。
    *   **缺口回補**：關注跳空缺口的回補機會。

### 2. 出場策略 (Exit)
*   **利多出貨**：消息見報大漲時出貨，「賺錢要簡單」。
*   **分批獲利**：常用「賣一半、留一半」策略，入袋為安，剩下的部位博取超額利潤。
*   **時效性**：若進場後 3 天內未如預期發動或過高，則換股操作，不跟股票談戀愛。
*   **放空機制**：對過度炒作、基本面不符的股票進行放空（如：中工、轉弱後的萬海）。

## 五、 總結：XMZ 成功密鑰
1.  **心態超脫**：不被市場情緒左右，反向利用市場情緒。
2.  **效率工具**：利用股期的高槓桿與低成本，最大化週轉率。
3.  **趨勢共振**：鎖定報價趨勢與投信動向，站在資金的浪頭上。
4.  **果斷執行**：利多出盡敢賣，看錯止損敢砍，買點不到敢等。
