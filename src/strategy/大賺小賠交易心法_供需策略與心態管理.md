## 🎯 一、交易策略尋找方法

**核心觀念：賺錢的方法你都知道，只是沒花時間轉換**

### 實戰案例

**1️⃣ 新春行情（過年行情）**
- 第一層思維：過年前買股票，年後賣出
- 第二層架構：選大型股/小型股？高價/低價？
- 執行方法：手動統計自選股過去10年過年走勢（50檔x10年=500個線圖，花1小時）

**2️⃣ 融資斷頭策略**
- 第一層：融資大減排行榜前十名
- 第二層：加上「維持率低於130%」條件
- 邏輯：系統性風險發生時，散戶斷頭後股價容易上漲

**3️⃣ 券商成交量爆發**
- 觀察：台股破兆元成交量（史上首次）
- 轉換：買券商股（只有利多沒利空，大不了平出）
- 結果：第三天盤中預估量達兆元，市場開始聯想

## ⚖️ 二、交易本質：供需不平衡

**核心公式：供給 vs 需求（P&Q圖）**

### 生活化案例

**🌪️ 颱風來臨 → 蔬菜供給減少**
- 賣菜的人：有庫存要先賣嗎？→ 等颱風來價格上漲再賣
- 沒菜的人：先去買一堆菜囤貨 → 等颱風來價格上漲再賣
- 關鍵：不用等新聞報導颱風警報，看氣象圖就可以提前佈局
- **風險報酬不對稱**：颱風沒來頂多平出，來了可能賺1倍

### 實戰案例：記憶體報價

**📈 2025年11月記憶體行情**
- 供需背景：DDR4轉DDR5/HBM（AI Server需求）
- 報價連漲：10/31起連續7天漲3-4%，累計漲30-40%

**關鍵轉折點**
- 11/13：報價突然從每天漲3-4% → 僅漲0.18%
- 股價反應：11點報價公布後，從漲停殺到-10%（短短幾分鐘）
- 交易策略：這不是技術分析能解釋的，是**資訊領先優勢**

**進階玩法**
- 11點報價公布後大戶開始買「10:50提前報價」（月費15萬）
- 講者提醒：現場3000人x15萬=4.5億，已經失效不要買了😂

**第二次轉折**
- 11/23：報價從連續趨緩(0-1%) → 突然漲3.55%
- 股價反應：從跌停急拉+5%
- 關鍵：化解「報價翻跌」疑慮，市場情緒逆轉

## 🧠 三、心態管理：大賺小賠的關鍵

### 核心觀念

**⚠️ 勝率提升有限，風報比可以無限大**
- 講者勝率：50% → 55%（花了10年才提升10%）
- 極限勝率：60-80%
- 但風報比可以：賠100萬賺1億 = 100倍

**💡 期望值 = 勝率 x 風報比**
→ 與其追求高勝率，不如提升風報比

### 實戰方法：大部位賺錢、小部位試錯

**問題診斷**
- 下單後覺得「不如預期」「早知道不要下」→ 這種感覺小部位也會有
- 解法：**用小部位去換感覺**

**操作原則**
1. 沒把握時 → 最小單位（1張）試單
2. 感覺對了（進場就漲/跌，覺得自己是股神）→ 大部位加碼
3. 感覺錯了 → 立刻停損，因為只有小部位不會痛

**關鍵突破**
- 講者到現在有時還是下1張試單
- 現場問「誰從1張開始下？」→ 沒人舉手
- 問題：**你花太多錢在嘗試上**

### 心態訓練：賠錢放大感受、賺錢縮小感受

**目標比例**
- 一般人：賠300元 = 賺3000元的感受（10倍）
- 高手：賠300元 = 賺30萬的感受（1000倍）
- 講者：我的倍數非常非常高

**訓練方法**

**✅ 賠錢時：**
- 賠1000元 → 告訴自己「世界末日」「徹底被市場淘汰」
- 跟小孩說「爸爸今天賠了你一罐奶粉錢」（讓小孩說「爸爸加油」）

**❌ 賺錢時：**
- 賺100萬 → 告訴自己「明天就要輸光了」
- 不要拿去吃好料，頂多「吃飯加一個蛋」
- 朋友問「今天賺多少？」→ 回答「哇幹沒做好」

**為什麼要這樣？**
- 讓你對小錢保持敏感 → 謹慎判斷行情
- 對大錢保持冷靜 → 不會因為「覺得自己是股神」而誤判

## ⚠️ 四、最常見的錯誤：一個行情兩種解讀

### 經典案例：南亞科從50元漲到250元回檔

**情境一：你50買、250賣出**
→ 回檔時想法：「好險我出場了，轉弱了」（偏空判斷）

**情境二：你200買、250套牢、回檔到200出場**
→ 回檔時想法：「好險及時出場沒賠錢，轉弱了」（偏空判斷）

**情境三：你50-250都沒買到，一直等回檔**
→ 回檔時想法：「強勢股終於有機會上車了！」（偏多判斷）

**情境四：你250買進，看外資喊目標價500**
→ 回檔時想法：「買太高了，拉回攤平！」（偏多判斷）

### 問題根源

**🔴 一樣的行情、一樣的價位，卻有不同解讀**
→ 要麼是你的**部位**出問題
→ 要麼是你的**資金**出問題
→ 要麼是你的**槓桿**出問題
→ 要麼是你的**心態**出問題

### 解決方法

**💡 行情強弱 ≠ 你的部位大小**
- 你下1張 vs 100張，都不會影響行情走勢
- 進場/出場判斷 ≠ 你的庫存損益
- 你等了多久、成本在哪裡 → 跟行情強弱無關

**講者作法**
- 下單畫面只顯示「現在幾張」
- 不顯示損益、不顯示均價
- 理由：避免被自己的部位影響判斷

**✅ 解決這四個問題，你就能看到市場真實的樣貌**

   - 記憶體報價案例不是「破均線」，是供需資訊改變

   - 不是有什麼武功秘笈
   - 只是想像力、觀察、經驗的差異

3. **「你跟大戶的距離只有經驗跟邏輯」**
   - 以及有沒有把經驗邏輯轉換成交易策略

4. **「賠錢的感覺放大、賺錢的感覺縮小 → 風報比就拉開了」**

5. **「有時候撐著人的不是毅力，而是那份單純的熱愛」**
