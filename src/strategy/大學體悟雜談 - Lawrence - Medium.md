# 大學體悟雜談

作者：Lawrence
發表日期：2024年8月30日
來源：https://medium.com/@lawrence910426

大學四年的努力大概可以濃縮成這一張簡歷，說很厲害好像也沒有，說很混好像也沒混到哪裡去，總之就是過了人生中的四年，但這四年確實過得滿有趣的 XD

![簡歷圖片](https://miro.medium.com/v2/resize:fit:1400/1*TIESImUw7QrHlBbzIdK2eA.png)

大學四年內，我花了不少時間在做交易，也花了不少時間在思考自己的人生。那麽，我就在這邊跟各位分享一下我對我自己的人生，還有做交易的體悟吧。

## 人生體悟

1.  跟別人內耗很蠢，自己過得爽最重要
2.  溝通的目的是取得共識，不可能取得共識的就別浪費時間了
3.  跟厲害的人混，能讓你少走好幾年彎路，他會跟你講容易成功的方法
4.  在你砸時間去努力前，搞清楚要怎麼努力比較有效，亂砸時間是很蠢的事
5.  與其花很大的努力去做事，不如用聰明的方法來做事
6.  很多時候並不是你不夠努力，只是單純別人比你有天賦
7.  完成任務比努力還重要，除非你的任務是裝忙
8.  很多人覺得拖鞋配短褲很邋遢，但是穿起來真的很舒服
9.  科技業的職場著裝指南 = 只要你不妨礙到他人就行
10.  工作的目的是為了賺錢，不是為了面子

## 交易體悟

1.  你不太可能是第一個發現策略的人，但是這並不妨礙去賺 alpha，雖然可能 alpha 比較少
2.  不一定要貴森森的機器才能低延遲，很多時候只要程式寫得沒這麼糟，就能比多數的 XQ 仔還快了
3.  處理一下 Latency，然後該 non-blocking 的地方 non-blocking，這樣就能比多數人的手眼協調還快了
4.  與其一直花時間做很多沒什麼效益的 trade，不如直接把開盤睡過去
5.  要多照顧一下自己的程式，不要直接發 cronjob，要不然賠了一堆都還沒發現
6.  螢幕上顯示的行情不一定是交易所內實際上的 order book，很有可能是一段時間前的某個 snapshot 而已
7.  停損不一定能為帶來優勢，但是能給我一種虛偽的安心感
8.  與其好好設計一個 sandbox 把策略放進去跑，不如直接用迷你倉上線去跑
9.  不一定要完全自動化的程式交易，只要把你做不到或是懶的做的部分丟給電腦就好
10.  有可重現性的技巧再去學就好，對於線仙我只能表達我的欽佩，並牢記自己並沒有這項天賦
11.  天底下有三件事不能相信，這單穩賺，我能凹贏，我比交易對手聰明
12.  不同人有不同的優勢，善用自己的優勢，保護好自己的劣勢
13.  論文很棒，儘管不能直接幫你賺錢，但是能夠給你很多 insight
14.  在程式碼鑽漏洞，你會被抓去 HITCON 演講；在市場中鑽交易規則的漏洞，你會發大財
15.  越厲害的人往往越認為自己很弱，這件事不論在競程圈或是交易圈都成立
16.  不要相信程式很穩，即使你的程式沒壞掉，你不能保證交易所不會壞掉
17.  短 trade 很累，都在賺辛苦錢
18.  聰明人懶的做，大戶不想做，散戶不能做，通常這就是不錯的策略了
19.  不要去亂挑戰交易所的底線，被警告會很麻煩
20.  很蠢的三件事：行情來了，但我睡過去了；考試來了，但我睡過去了；早八體育，但我睡過去了
21.  多跟贏家混，少跟輸家混，贏家的 insight 比輸家有用很多
22.  如果有人能穩定獲利，那他是天才；如果有人扣掉稅金還能穩定賠錢，那他是鬼才
23.  不使用量化方法就賺大錢的人很多，但用量化方法賺小錢比較容易
24.  回測固然重要，但是能搞清楚自己在賺什麼錢更重要
25.  人工交易的好處在於能做很多臨場判斷，程式交易的好處在於能搶快
26.  人工交易很麻煩，但是對於流動性差的商品有其必要性
27.  技術分析不是沒有用，只是他能提供的 alpha 很可能微乎其微
28.  遇到會攻擊你說「只有指數投資能賺錢」的人，去賺他的錢就好，不用花時間跟他吵架
29.  羨慕別人的對帳單很蠢，但是每次看到別人爆賺總會覺得自己是笨蛋
30.  把成交量刷高很重要，這樣才能拿到各式各樣的優勢
31.  對帳單就跟自己的日記一樣，都能看到之前幹了多少蠢事
32.  即使有再聰明的腦袋，沒有完整的資料集也不能做研究
33.  越簡單的策略，越容易有驚人的績效；越複雜的策略，越難分析自己在賺什麼錢
34.  在不超過 Kelly 的前提下，決定開多大槓桿是一門藝術
35.  事件型策略的 event distribution 越不平均，實際交易起來就越容易手忙腳亂
36.  看到回測績效很漂亮時，記得多檢查一下，很可能沒辦法實際交易出這個績效
37.  不要輕易為了做交易辭去正職工作，多數情況下不會划算
38.  不要相信市場很有效率，往往有些沒人注意到的地方充滿機會
39.  機器學習很棒，但提取有用的特徵更重要
40.  不要被工具侷限了你的想法，要善用工具去實踐你的想法
41.  在開始研究之前，先檢查一下你的資料來源乾不乾淨，不然可能會做白工
42.  用運氣賺來的錢，遲早會用實力賠回去
43.  有事沒事可以去盯一下盤，搞不好洗澡時會突然蹦出 insight
44.  自信心膨脹而加大槓桿時，通常都是爆炸的開端
45.  不一定要是最聰明的才能獲利，只要少做蠢事就行了
46.  多數回測的 code 都能叫 ChatGPT 幫你寫，但是怎麼回測必須要自己動腦想
47.  交易的世界也有類似 PPA (Power/Performance/Area) 的限制，大概是 RRC (Return/Risk/Capital Size)
48.  Overtrade 是一件很糟糕的事情，一來是他會浪費你的時間，二來是他會浪費你的錢

## 結語

收到兵單了，接下來就是等當兵了，很快的人生又過了一個里程碑了，在軍中有看到什麼酷東西再來跟大家分享！
