## 🚀 使用指南

### 快速開始
```bash
# 完整展示
./full_demo.sh

# 或分步執行
make test-local-ffi    # 本地 FFI 測試
make run               # 啟動應用程式
make logcat           # 監控日誌
```

### 日常開發
```bash
make dev              # 開發模式 (熱重載)
make test             # 執行所有測試
make check            # 程式碼品質檢查
make format           # 程式碼格式化
```

### Android 測試
```bash
# 連接 Android 設備後
make test-android     # 在設備上測試
make test-with-logs   # 同時監控日誌
```
