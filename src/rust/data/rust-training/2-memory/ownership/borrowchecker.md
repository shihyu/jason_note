借用是存在風險的

如下面的這個借用，x的內存生命週期提前結果被釋放，但是r仍然在引用x的內存，這就會存在問題。

```
fn main() {
    let r;                  // ---------+-- 'a
                            //          |
    {                       //          |
    let x = 5;              // -+-- 'b  |
    r = &x;                 //  |       |
    }                       // -+       |
                            //          |
    println!("r: {}", r);   //          |
}
```

所以，編譯時的借用檢查器會阻止這樣的代碼。編譯時會報錯  

```shell
error[E0597]: `x` does not live long enough
```
