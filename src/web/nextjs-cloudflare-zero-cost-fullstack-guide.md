## зӮәд»ҖйәјиӘӘгҖҢдёҖеҖӢдәәе°ұиғҪжҗһе®ҡгҖҚпјҹ

иӘӘгҖҢдёҖеҖӢдәәе°ұиғҪжҗһгҖҚпјҢдёҚжҳҜеҗ№зүӣгҖӮж ёеҝғжҳҜйҖҷеҖӢжЁЎжқҝе·Із¶“жҠҠжңҖз№Ғз‘Јзҡ„гҖҢй«’жҙ»зҙҜжҙ»гҖҚе№№е®ҢдәҶгҖӮ

дҪ дёҚз”ЁеҶҚеҫһйӣ¶й–Ӣе§Ӣжҗӯжһ¶еӯҗгҖӮйҖҷеҖӢжЁЎжқҝжҠҠдёҖеҖӢе…Ёз«ҜжҮүз”ЁйңҖиҰҒзҡ„жқұиҘҝйғҪеҲҶеҘҪйЎһпјҢдёҰжҸҗдҫӣдәҶжңҖдҪіеҜҰиёҗпјҡ

- **еүҚз«Ҝпјҡ** Next.js 15.5.2пјҲApp Router + TypeScriptпјү
- **еҫҢз«Ҝпјҡ** Cloudflare Pages FunctionsпјҲEdge RuntimeпјүпјҢAPI и·Ҝз”ұй–Ӣз®ұеҚіз”Ё
- **иіҮж–ҷпјҡ** Prisma ORM + D1 иіҮж–ҷеә«
- **е„Іеӯҳпјҡ** R2 зү©д»¶е„Іеӯҳ
- **еҝ«еҸ–пјҡ** KV йҚөеҖје„Іеӯҳ
- **и§Җжё¬пјҡ** зөҗж§ӢеҢ–ж—ҘиӘҢ + Analytics дәӢд»¶
- **е·ҘзЁӢпјҡ** зөұдёҖеӣһжҮүпјҸйҢҜиӘӨпјҸдёӯд»Ӣи»ҹй«”пјҸйҖҹзҺҮйҷҗеҲ¶
- **жё¬и©Ұпјҡ** Vitest
- **йғЁзҪІпјҡ** Wrangler + GitHub Actions

дҪ иғҪзңҹжӯЈе°ҲжіЁж–јжҘӯеӢҷжң¬иә«пјҡй ҒйқўгҖҒиіҮж–ҷжЁЎеһӢгҖҒд»ӢйқўйӮҸијҜгҖӮе…¶йӨҳзҡ„е·ҘзЁӢеҝ…йңҖе“ҒпјҢдҫӢеҰӮж—ҘиӘҢгҖҒйҷҗжөҒгҖҒйҢҜиӘӨиҷ•зҗҶгҖҒеӣһжҮүж јејҸгҖҒеҝ«еҸ–гҖҒдёҠеӮігҖҒеҒҘеә·жӘўжҹҘгҖҒеӨҡз’°еўғй…ҚзҪ®иҲҮ CI/CDпјҢйғҪе·Ій җиЁӯжҸҗдҫӣгҖӮ

е°ҚдҪ дҫҶиӘӘпјҢйҖҷж„Ҹе‘іи‘—пјҡ

1. **зҜҖзңҒеӨ§йҮҸжҷӮй–“пјҡ** жёӣе°‘ 70% д»ҘдёҠзҡ„йҮҚиӨҮй…ҚзҪ®е·ҘдҪңгҖӮ
2. **дёҚз”ЁзіҫзөҗйҒёеһӢпјҡ** зӣҙжҺҘдҪҝз”Ё Next.js + Cloudflare йҖҷеҘ—зөұдёҖзҡ„зҸҫд»ЈжҠҖиЎ“жЈ§гҖӮ
3. **йғЁзҪІз„ЎжҶӮпјҡ** з’°еўғи®Ҡж•ёгҖҒCI/CD и…іжң¬иҲҮж–Үд»¶йғҪе·ІеӮҷеҘҪпјҢз…§и‘—и·‘е°ұиЎҢгҖӮ

## иј•й¬ҶдёҠжүӢпјҡ5 жӯҘи·‘йҖҡе…Ёз«Ҝ

дёҠжүӢдёҚйӣЈпјҢйҖҷеҖӢжЁЎжқҝжҠҠй—ңйҚөжӯҘй©ҹйғҪи…іжң¬еҢ–дәҶгҖӮдҪ еҸӘйңҖиҰҒжә–еӮҷеҘҪйҖҷе№ҫжЁЈжқұиҘҝпјҡ

- Node.js >= 20пјҲе°ҲжЎҲиЈЎжңү `.nvmrc`пјү
- pnpm >= 8
- дёҖеҖӢ Cloudflare еёіжҲ¶пјҲйңҖиҰҒй–ӢйҖҡ PagesгҖҒWorkersгҖҒD1гҖҒR2гҖҒKVпјү

з„¶еҫҢпјҢи·ҹи‘— `QUICKSTART-zh.md` ж–Үд»¶пјҢ5 жӯҘе°ұиғҪи·‘иө·дҫҶпјҡ

1. **еҲқе§ӢеҢ–пјҡ** `pnpm i` иЈқеҘҪдҫқиіҙпјҢ`wrangler login` зҷ»е…Ҙ CloudflareгҖӮ
2. **й…Қз’°еўғпјҡ** й…ҚзҪ® `wrangler.toml`пјҲжң¬ең°пјүгҖҒ`wrangler.test.toml`пјҲжё¬и©Ұпјүе’Ң `wrangler.prod.toml`пјҲжӯЈејҸз’°еўғпјүгҖӮ
3. **е»әиіҮжәҗпјҡ**
   - D1пјҲиіҮж–ҷеә«пјүпјҡ`pnpm run db:migrate:local`
   - R2пјҲе„Іеӯҳпјүпјҡ`pnpm run r2:create:test`пјҲжҲ– `prod`пјү
   - KVпјҲеҝ«еҸ–пјүпјҡ`pnpm run kv:create:test`пјҲжҲ– `prod`пјү
4. **и·‘й–Ӣзҷјпјҡ**
   - жң¬ең°й–ӢзҷјпјҲNext.jsпјүпјҡ`pnpm dev`
   - жЁЎж“¬ Cloudflare з’°еўғпјҡ`pnpm run cf:dev`
5. **йғЁзҪІдёҠз·ҡпјҡ**
   - жё¬и©Ұз’°еўғпјҡ`pnpm run pages:deploy:test`
   - жӯЈејҸз’°еўғпјҡ`pnpm run pages:deploy:prod`

еҰӮжһңй ҶеҲ©пјҢдҪ дёҖеӨ©е…§е°ұиғҪи·‘йҖҡ DemoпјҢдёҖйҖұе…§е°ұиғҪдёҠз·ҡ MVPпјҢеҺ»й©—иӯүжө·еӨ–иЁӘе•Ҹй«”й©—гҖӮ

## йӣ¶жҲҗжң¬иө·жӯҘпјҡCloudflare е…ҚиІ»йЎҚеәҰжңүеӨҡйҰҷпјҹ

гҖҢе®Ңе…Ёе…ҚиІ»гҖҚжҳҜйҖҷеҘ—ж–№жЎҲжңҖеӨ§зҡ„иӘҳжғ‘гҖӮCloudflare зҡ„ Free Tier зңҹзҡ„йқһеёёеӨ§ж–№пјҢз”ЁдҫҶеҒҡ MVP з”ҡиҮідёӯе°ҸжөҒйҮҸе°ҲжЎҲйғҪз¶Ҫз¶ҪжңүйӨҳпјҢдҪҶеҜҰйҡӣйЎҚеәҰд»ҚиҰҒд»Ҙ Cloudflare е®ҳж–№жңҖж–°ж–Үд»¶зӮәжә–гҖӮ

- **PagesпјҲйқңж…ӢиЁ—з®Ўпјү**
  - е°ҲжЎҲж•ёйҮҸпјҡ100 еҖӢ
  - жҜҸжңҲе»әзҪ®пјҡ500 ж¬Ў
  - й »еҜ¬пјҸйқңж…Ӣи«ӢжұӮпјҡз„ЎйҷҗеҲ¶
- **Pages FunctionsпјҲиҲҮ Workers е…ұдә«йЎҚеәҰпјү**
  - жҜҸж—Ҙи«ӢжұӮпјҡ100,000 ж¬Ў
  - CPU жҷӮй–“пјҡжҜҸж¬Ўи«ӢжұӮ 10 жҜ«з§’
- **D1 DatabaseпјҲиіҮж–ҷеә«пјү**
  - иіҮж–ҷеә«ж•ёйҮҸпјҡ10 еҖӢ
  - зёҪе„Іеӯҳпјҡ5 GBпјҲжүҖжңүиіҮж–ҷеә«е…ұдә«пјү
- **R2 StorageпјҲзү©д»¶е„Іеӯҳпјү**
  - **ж ёеҝғе„ӘеӢўпјҡйӣ¶еҮәз«ҷиІ»з”ЁпјҲEgress Zeroпјү**пјҢйҖҷй»һе°ҚеҮәжө·йқһеёёйҮҚиҰҒгҖӮ

йҖҷдәӣйЎҚеәҰе°ҚеҖӢдәәй–ӢзҷјиҖ…зӣёз•¶еҸӢе–„гҖӮдҪ иғҪд»Ҙ**йӣ¶дјәжңҚеҷЁжҲҗжң¬**и·‘дёҖеҖӢзңҹеҜҰзҡ„е…Ёз«ҜжҮүз”ЁгҖӮ

## Cloudflare еүҚеҫҢз«Ҝжһ¶иЁӯжңҖдҪіеҜҰиёҗ

еңЁ Cloudflare жһ¶иЁӯеүҚеҫҢз«Ҝз¶Із«ҷпјҢжңҖдҪіеҜҰиёҗжҳҜпјҡ**з”Ё Cloudflare Pages иЁ—з®ЎеүҚз«ҜпјҲReact / Vue / Astro зӯүпјүпјҢз”Ё Pages Functions жҲ– Cloudflare Workers иҷ•зҗҶеҫҢз«Ҝ API**пјҢйҒ”жҲҗе…ҚдјәжңҚеҷЁгҖҒе…Ёеҹҹ CDN еҠ йҖҹгҖҒе…ҚиІ»зҡ„й«ҳж•ҲиғҪжһ¶ж§ӢгҖӮ

### еүҚеҫҢз«Ҝжһ¶иЁӯжӯҘй©ҹ

#### жӯҘй©ҹ 1пјҡжә–еӮҷд»Јзўј

е°ҮеүҚз«Ҝе°ҲжЎҲ push еҲ° GitHub жҲ– GitLabгҖӮ

```bash
git add .
git commit -m "init project"
git push origin main
```

#### жӯҘй©ҹ 2пјҡйғЁзҪІеүҚз«ҜпјҲPagesпјү

1. зҷ»е…Ҙ CloudflareпјҢйҖІе…ҘгҖҢ**Workers & Pages**гҖҚ>гҖҢ**Create application**гҖҚ>гҖҢ**Pages**гҖҚ>гҖҢ**Connect to Git**гҖҚ
2. йҒёж“ҮдҪ зҡ„ Git е°ҲжЎҲ
3. дҫқжЎҶжһ¶иЁӯе®ҡе»әж§ӢжҢҮд»ӨиҲҮијёеҮәзӣ®йҢ„пјҡ

| жЎҶжһ¶ | е»әж§ӢжҢҮд»Ө | ијёеҮәзӣ®йҢ„ |
| --- | --- | --- |
| React (Vite) | `npm run build` | `dist` |
| Next.js | `npm run build` | `.next` / `out` |
| Vue | `npm run build` | `dist` |
| Astro | `npm run build` | `dist` |
| SvelteKit | `npm run build` | `build` |

4. й»һж“ҠгҖҢ**Save and Deploy**гҖҚпјҢCloudflare иҮӘеӢ•е»әзҪ®дёҰйғЁзҪІгҖӮ

#### жӯҘй©ҹ 3пјҡй…ҚзҪ®еҫҢз«ҜпјҲе…©зЁ®ж–№жі•пјү

**ж–№жі• AпјҲжҺЁи–ҰпјүвҖ” Pages Functions**

еңЁе°ҲжЎҲж №зӣ®йҢ„е»әз«Ӣ `/functions` иіҮж–ҷеӨҫпјҢжӘ”жЎҲеҚіи·Ҝз”ұпјҢйҡЁеүҚз«ҜдёҖиө·йғЁзҪІпјҡ

```text
my-project/
в”ңв”Җв”Җ src/               вҶҗ еүҚз«Ҝжәҗзўј
в”ңв”Җв”Җ functions/
в”Ӯ   в”ңв”Җв”Җ api/
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ hello.ts   вҶ’ GET /api/hello
в”Ӯ   в”Ӯ   в””в”Җв”Җ users/
в”Ӯ   в”Ӯ       в””в”Җв”Җ index.ts вҶ’ GET/POST /api/users
в”Ӯ   в””в”Җв”Җ _middleware.ts вҶҗ е…Ёеҹҹдёӯд»Ӣи»ҹй«”
в”ңв”Җв”Җ package.json
в””в”Җв”Җ wrangler.toml
```

```typescript
// functions/api/hello.ts
export const onRequest: PagesFunction = async (ctx) => {
  return Response.json({ message: "Hello from Edge!" });
};
```

**ж–№жі• B вҖ” зҚЁз«Ӣ Cloudflare Workers**

йҒ©з”Ёж–јйңҖиҰҒиҲҮеүҚз«ҜеҲҶйӣўйғЁзҪІзҡ„зҚЁз«ӢеҫҢз«ҜжңҚеӢҷпјҡ

1. **зөұдёҖ API еӣһжҮүиҲҮдёӯд»Ӣи»ҹй«”**
   - `lib/api/response.ts` е®ҡзҫ© `success`пјҸ`error` зҡ„зөҗж§ӢпјҢ`errorResponse` жңғиҮӘеӢ•жҳ е°„йҢҜиӘӨйЎһеһӢиҲҮзӢҖж…ӢзўјгҖӮ
   - `lib/api/middleware.ts` жңғиҮӘеӢ•жіЁе…Ҙ `requestId`гҖҒиЁҳйҢ„и«ӢжұӮиҖ—жҷӮпјҢдёҰиҝҪеҠ иҝҪи№ӨжЁҷй ӯгҖӮ

2. **йҢҜиӘӨйЎһеһӢй«”зі»**
   - `lib/errors/index.ts` е®ҡзҫ©еҚҒеӨҡзЁ®еёёиҰӢйҢҜиӘӨйЎһеһӢпјҢдҫӢеҰӮиіҮж–ҷеә«гҖҒй©—иӯүгҖҒиӘҚиӯүгҖҒйҷҗжөҒзӯүгҖӮ
   - `ERROR_STATUS_MAP` зөұдёҖз®ЎзҗҶ HTTP зӢҖж…ӢзўјпјҢи®“йҢҜиӘӨиҷ•зҗҶжӣҙиҰҸзҜ„гҖӮ

3. **зөҗж§ӢеҢ–ж—ҘиӘҢпјҲжӯЈејҸз’°еўғ JSONгҖҒй–Ӣзҷјз’°еўғ Prettyпјү**
   - `lib/logger/index.ts` ж”ҜжҸҙ `http()`гҖҒ`performance()`гҖҒ`query()` зӯүе°Ҳз”Ёж–№жі•пјҢд№ҹиғҪй…ҚзҪ®ж…ўжҹҘи©ўй–ҖжӘ»гҖӮ

4. **Analytics дәӢд»¶жү“й»һ**
   - `lib/analytics/index.ts` е®ҡзҫ©еҗ„зЁ®жҘӯеӢҷдәӢд»¶пјҢеҸҜйҒёж“Ү SinkпјҲдҫӢеҰӮеҜ«е…Ҙ D1 жҲ– KVпјүпјҢеӨұж•—жҷӮиҮӘеӢ•йҷҚзҙҡеҲ°ж—ҘиӘҢгҖӮ

5. **йҖҹзҺҮйҷҗеҲ¶пјҲKV ж»‘еӢ•иҰ–зӘ—пјү**
   - `lib/api/rate-limit.ts` еҲ©з”Ё KV зҡ„еҺҹеӯҗжҖ§еҜҰдҪңж»‘еӢ•иҰ–зӘ—йҷҗжөҒгҖӮй җиЁӯжҜҸеҲҶйҗҳ 300 ж¬ЎпјҢд№ҹиғҪеңЁи·Ҝз”ұдёҠиҮӘиЁӮпјҢдҫӢеҰӮйҷҗеҲ¶е»әз«ӢдҪҝз”ЁиҖ…д»ӢйқўзӮә `10/min`гҖӮ

6. **Repository жЁЎејҸ + Prisma + D1 е–®дҫӢ**
   - `repositories/*` е°ҒиЈқиіҮж–ҷеӯҳеҸ–пјҢ`lib/api/database.ts` жҸҗдҫӣ `withRepositories` жіЁе…ҘгҖӮ
   - `lib/db/client.ts` е°Қ Prisma Client еҒҡдәҶ**е–®дҫӢйҮҚз”Ё**гҖӮйҖҷеҫҲй—ңйҚөпјҢиғҪжёӣе°‘зҙ„ 50 еҲ° 100 жҜ«з§’зҡ„иіҮж–ҷеә«еҶ·е•ҹеӢ•й–ӢйҠ·гҖӮ

7. **й–Ӣз®ұеҚіз”Ёзҡ„еҝ«еҸ–иЈқйЈҫеҷЁ**
   - `lib/cache/client.ts` жҸҗдҫӣ `withCache(key, fn, ttl)` иЈқйЈҫеҷЁпјҢиҮӘеӢ•иҷ•зҗҶеҝ«еҸ–е‘ҪдёӯгҖҒз©ҝйҖҸиҲҮеӣһжәҗгҖӮ

8. **е°ҒиЈқ R2 дёҠеӮіиҲҮдёӢијү**
   - `lib/r2/client.ts` е°ҒиЈқдёҠеӮіиҲҮдёӢијүпјҢ`app/api/upload/route.ts` жҳҜе®Ңж•ҙзҜ„дҫӢпјҢдҪ дёҚйңҖиҰҒзӣҙжҺҘйқўе°ҚиӨҮйӣңзҡ„ S3 SDKгҖӮ

9. **еҒҘеә·жӘўжҹҘз«Ҝй»һ**

## еҜҰиёҗи·Ҝеҫ‘пјҡеҫһйӣ¶еҲ° MVP

- йҖҸйҒҺ `logger.query`гҖҒ`performance` иҲҮ Analytics и§Җжё¬ж…ўжҹҘи©ўиҲҮж…ўж“ҚдҪңгҖӮ
- ж”¶йӣҶжө·еӨ–дҪҝз”ЁиҖ…еӣһйҘӢпјҢеҝ«йҖҹиҝӯд»ЈгҖӮ

## жҲҗжң¬иҲҮж“ҙеұ•пјҡе…ҚиІ»иө·жӯҘпјҢе№іж»‘еҚҮзҙҡ

**е…ҚиІ»еұӨиғҪи·‘еӨҡд№…пјҹ**

дҫқдҪңиҖ…з¶“й©—пјҢеҒҡ MVP е’Ңдёӯе°ҸжөҒйҮҸпјҲж—Ҙ 10 иҗ¬и«ӢжұӮпјүзө•е°ҚеӨ з”ЁгҖӮ5 GB зҡ„ D1 е„Іеӯҳд№ҹиғҪж”Ҝж’җдёҖж®өжҷӮй–“пјҢиҖҢ R2 зҡ„йӣ¶еҮәз«ҷиІ»жӣҙжҳҜе·ЁеӨ§зҡ„жҲҗжң¬е„ӘеӢўгҖӮ

**жңӘдҫҶеҰӮдҪ•ж“ҙеұ•пјҹ**

з•¶дҪ зҡ„жҘӯеӢҷй–Ӣе§ӢжҲҗй•·еҫҢпјҢйҖҷеҘ—жһ¶ж§Ӣд№ҹиғҪе№іж»‘еҚҮзҙҡпјҡ

1. **иіҮж–ҷпјҡ** и®ҖеҜ«еЈ“еҠӣи®ҠеӨ§жҷӮпјҢе…ҲдёҠи®Җеҝ«еҸ–пјҲKVпјүпјӣеҶҚдёҚеӨ е°ұиҖғж…®жҺҘе…Ҙе°ҲжҘӯиЁ—з®ЎиіҮж–ҷеә«пјҢдҫӢеҰӮ PlanetScaleпјҢи®“ Edge жү®жј” BFF еұӨгҖӮ
2. **е„Іеӯҳпјҡ** R2 жң¬иә«ж•ҲиғҪеҫҲеј·пјҢеҫҢзәҢйҮҚй»һйҖҡеёёеңЁз®ЎзҗҶзӯ–з•ҘгҖӮ
3. **и§Җжё¬пјҡ** Analytics Engine еҸҜд»ҘжҢҒд№…еҢ–иіҮж–ҷпјҢд№ҹиғҪеҶҚе°ҚжҺҘеӨ–йғЁеҸҜи§Җжё¬е№іеҸ°гҖӮ

## еёёиҰӢе•ҸйЎҢпјҲFAQпјү

**Q1пјҡзӮәд»ҖйәјйҒё Next.js + Cloudflareпјҹ**  
AпјҡеүҚз«Ҝз”ҹж…ӢжҲҗзҶҹпјҢеҠ дёҠ Edge еҺҹз”ҹеҹ·иЎҢжҷӮгҖӮе°ҚзҚЁз«Ӣй–ӢзҷјиҖ…дҫҶиӘӘпјҢжІ’жңүдјәжңҚеҷЁз¶ӯиӯ·жҲҗжң¬гҖҒе…Ёзҗғ POP е„ӘеӢўпјҢд»ҘеҸҠ D1гҖҒR2гҖҒKV зҡ„дёҖй«”еҢ–пјҢжҳҜеҫҲй—ңйҚөзҡ„жұәе®ҡеӣ зҙ гҖӮ

**Q2пјҡжңғдёҚжңғеҫҲйӣЈдёҠжүӢпјҹ**  
AпјҡеҰӮжһңдҪ жң¬дҫҶе°ұзҶҹ Next.jsпјҢе№ҫд№ҺжІ’жңүй–ҖжӘ»гҖӮжЁЎжқҝе·Із¶“жҠҠиіҮж–ҷеә«йҖЈжҺҘгҖҒж—ҘиӘҢгҖҒйғЁзҪІйҖҷдәӣжңҖйә»з…©зҡ„е·ҘзЁӢйғЁеҲҶеҢ…еҘҪпјҢдҪ еҸӘйңҖиҰҒеҜ«жҘӯеӢҷгҖӮ

**Q3пјҡе…ҚиІ»еұӨзңҹзҡ„еӨ з”Ёе—Һпјҹ**  
Aпјҡе°Қ MVP е’Ңдёӯе°ҸжөҒйҮҸдҫҶиӘӘйҖҡеёёи¶іеӨ гҖӮдёҠз·ҡеҫҢжҢҒзәҢи§ҖеҜҹ Cloudflare з”ЁйҮҸзөұиЁҲпјҢеҝ«и¶…йЎҚжҷӮеҶҚеҚҮзҙҡжҲ–е„ӘеҢ–еҚіеҸҜгҖӮ

**Q4пјҡеҰӮдҪ•йқўеҗ‘жө·еӨ–еҒҡе„ӘеҢ–пјҹ**  
Aпјҡж ёеҝғе°ұжҳҜзӣЎйҮҸйқ иҝ‘дҪҝз”ЁиҖ…гҖӮе–„з”ЁйӮҠз·Јеҝ«еҸ–пјҲKVпјүгҖҒR2 йӣ¶еҮәз«ҷиІ»пјҢд»ҘеҸҠ SSRгҖҒSSGгҖҒISRпјҢжёӣе°‘и·ЁеҚҖеҫҖиҝ”пјӣеҗҢжҷӮзӮәд»ӢйқўеҠ дёҠ `withCache` е’ҢеҗҲзҗҶзҡ„ `Cache-Control` жЁҷй ӯгҖӮ

**Q5пјҡеҰӮдҪ•ж“ҙеұ•иӨҮйӣңжҘӯеӢҷпјҹ**  
AпјҡйҖҸйҒҺ Repository жЁЎејҸж“ҙеұ•иіҮж–ҷеӯҳеҸ–еұӨпјӣеңЁ API еұӨзөұдёҖдҪҝз”Ё `withRepositories` + `withMiddleware`гҖӮPrisma жң¬иә«д№ҹж”ҜжҸҙдәӨжҳ“гҖҒеҲҶй ҒиҲҮйҒҺжҝҫзӯүиғҪеҠӣгҖӮ
