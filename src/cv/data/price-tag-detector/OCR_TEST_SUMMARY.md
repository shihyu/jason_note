## 🔧 技術改進

### 1. 針對性預處理

#### 黃色區域（黑字黃底）
- **策略 1**: CLAHE + 反色 + 自適應二值化
- **策略 2**: 簡單反色 + Otsu 二值化
- **策略 3**: 對比度增強 + 閾值二值化

#### 綠色區域（白字綠底）
- **策略 1**: CLAHE + 自適應二值化
- **策略 2**: Otsu 二值化
- **策略 3**: 亮度增強 + 閾值二值化

### 2. 多策略識別

```
3 種預處理方法 × 4 種 Tesseract 配置 = 12 種組合
自動選擇最佳結果
```

### 3. 字符白名單

```python
tessedit_char_whitelist=0123456789$.,/KMTBs
```

只允許價格相關字符，減少誤識別。

### 4. 智能文字清理

```python
常見錯誤修正:
'S' → '$'  # S 誤認為 $
'l' → '1'  # l 誤認為 1
'O' → '0'  # O 誤認為 0
'|' → '1'  # | 誤認為 1
```

### 5. 格式驗證

支持的價格格式:
- `$123` - 簡單價格
- `$1.5` - 小數價格
- `$40K` - 千位單位
- `$125/s` - 每秒價格
- `$1.5M` - 百萬單位

## 🎓 技術亮點
