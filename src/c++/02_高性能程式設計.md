## 💡 性能優化技巧

### CPU 層級優化
1. **Cache 友善設計**
   - 數據局部性
   - Cache line 對齊
   - 避免 False Sharing

2. **減少分支預測失敗**
   - 使用 likely/unlikely
   - 分支重排序
   - 查表代替條件

3. **SIMD 向量化**
   - 自動向量化
   - 手動 SIMD 指令
   - 批次處理

### 記憶體優化
1. **記憶體池**
   - 減少 malloc/free
   - 預分配資源
   - 對象池模式

2. **對齊與填充**
   - 結構體對齊
   - Cache line 填充
   - 避免偽共享

### 並發優化
1. **無鎖數據結構**
   - CAS 操作
   - Lock-free queue
   - Memory ordering

2. **減少競爭**
   - 讀寫鎖
   - 細粒度鎖
   - 無鎖算法
