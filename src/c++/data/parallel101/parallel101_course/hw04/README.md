## 評分規則

- 在你的電腦上加速了多少倍，就是多少分！請在 PR 描述中寫明加速前後的用時數據。
- 最好詳細解釋一下為什麼這樣可以優化。會額外以乘法的形式加分。
- 比如你優化後加速了 50 倍，講的很詳細，所以分數乘 2，變成 100 分！
- 比如你優化後加速了 1000 倍，但是你的 PR 描述是空，所以分數乘 0，變成 0 分！
