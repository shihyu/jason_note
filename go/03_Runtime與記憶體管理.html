💡 GC 原理詳解

三色標記法

  1. 白色

    • 未訪問的對象
    • 可能被回收

  2. 灰色

    • 已訪問，但子對象未掃描
    • 工作隊列

  3. 黑色

    • 已訪問且子對象已掃描
    • 不會被回收

GC 階段

  1. Mark Setup (STW)

    • 開啟寫屏障
    • 準備標記

  2. Marking (併發)

    • 併發標記
    • 三色標記

  3. Mark Termination (STW)

    • 完成標記
    • 關閉寫屏障

  4. Sweep (併發)

    • 併發清除
    • 回收記憶體
