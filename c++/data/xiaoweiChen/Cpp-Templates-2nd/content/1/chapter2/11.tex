
\begin{itemize}
\item 
类模板是在实现时保留一个或多个类型参数的类。

\item 
要使用类模板，需要将类型作为模板参数传递。并为这些类型，实例化(并编译)类模板。

\item 
对于类模板，只有调用的成员函数会实例化。

\item 
可以为某些类型特化类模板。

\item 
可以偏特化某些类型的类模板。

\item 
C++17后，可以从构造函数中自动推导出类模板的参数。

\item 
可以定义聚合类模板。

\item 
若声明为按值调用，则模板类型的调用参数会衰变。

\item 
模板只能在全局/命名空间作用域或类声明内部声明和定义。
\end{itemize}
